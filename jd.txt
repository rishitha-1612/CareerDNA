Job Title: Machine Learning Developer

Company: TechCorp AI Solutions
Location: Bangalore, India (Hybrid)

About the Role:

- Natural Language Processing — BERT, transformers, HuggingFace (2+ years)
- SQL and data manipulation with Pandas and NumPy
- REST API development using FastAPI or Flask
- Docker and containerization
- Git and version control
- Data pipelines and ETL workflows

Good to Have:
- MLOps experience — MLflow, model monitoring, feature stores
- Apache Spark or PySpark for large scale data
- Kubernetes and cloud platforms (AWS or GCP)
- CI/CD pipelines with GitHub Actions

Qualifications:
- Bachelor's or Master's degree in Computer Science, Data Science, or related field
- 2-4 years of experience in machine learning or AI development
- Strong problem solving and analytical thinking skills
- Good communication and agile/scrum experience

Salary: 8-15 LPA depending on experience
